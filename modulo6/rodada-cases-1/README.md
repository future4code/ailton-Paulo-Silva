# Jogo de Tarot - Challenge Personare

### Surge: https://macabre-zephyr.surge.sh/

### Tecnologias usadas

- JavaScript
- React JS
- HTML
- CSS

### Como executar o projeto:

Clone o repositório com o link:

```
https://github.com/
```

Instale as dependências com :

```
npm install
```

Inicie a aplicação com:

```
 npm run start
```
